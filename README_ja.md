# SentinelAI

SentinelAIは、AIを活用したセキュリティおよびオブザーバビリティWebアプリケーションを開発する、長期的なポートフォリオ兼プロダクト開発プロジェクトです。

Splunk、Grafana、ChatGPT、Claude、ローカルLLMなどの製品や技術を参考にしています。

## プロジェクトビジョン

SentinelAIは、エンジニア、セキュリティ担当者、小規模組織が、システムログやセキュリティログを収集、検索、分析、理解できる仕組みを提供することを目指します。

将来的には、以下の機能を実装する予定です。

* ログ収集
* ログ検索およびフィルタリング
* セキュリティイベントの可視化
* 機密情報のマスキング
* AIによるログ説明
* インシデント要約
* レポート生成
* アラート
* システム監視
* セルフホスト対応
* SaaS対応

## Phase 1の対象範囲

Phase 1では以下を実装します。

* ユーザー登録
* ログイン
* JWT認証
* ダッシュボード画面
* ログアウト
* PostgreSQLユーザーテーブル
* Docker Compose開発環境
* バックエンドテスト
* 基本ドキュメント

Phase 1では以下を実装しません。

* AI連携
* ログアップロード
* ログ検索
* 機密情報マスキング
* アラート
* 監視
* AWSデプロイ
* Kubernetes
* RBAC
* エンタープライズ機能

## 開発方針

SentinelAIは、実際のプロダクト開発と同様に段階的に開発します。

以下を重視します。

* セキュリティ
* 保守性
* クリーンアーキテクチャ
* 明確なドキュメント
* テスト可能性
* 適切なGit運用
* 実運用を意識した設計

開発速度よりも、ソフトウェア品質を優先します。
